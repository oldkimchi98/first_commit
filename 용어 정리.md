"문자열1" + "문자열2" : 문자열을 연결합니다.

"문자열" \* n : 문자열을 n번 반복합니다.

len("문자열") : 문자열의 길이를 반환합니다.

문자열\[start:stop] : start부터 stop 앞까지 문자열을 잘라 반환합니다.

문자열\[start:stop:step] : start부터 stop 앞까지 step씩 건너뛰며 문자열을 반환합니다.

split(delimiter) : delimiter를 기준으로 문자열을 리스트로 분리합니다.

delimiter.join(리스트) : 리스트의 문자열을 delimiter로 연결하여 하나의 문자열로 만듭니다.

capitalize() : 첫 글자는 대문자, 나머지는 소문자로 변환합니다.

upper() : 모든 문자를 대문자로 변환합니다.

lower() : 모든 문자를 소문자로 변환합니다.

startswith() : 지정한 문자열로 시작하면 True를 반환합니다.

endswith() : 지정한 문자열로 끝나면 True를 반환합니다.

find() : 문자열의 위치(인덱스)를 반환합니다. 없으면 -1을 반환합니다.

index() : 문자열의 위치(인덱스)를 반환합니다. 없으면 오류가 발생합니다.

isalnum() : 영문자 또는 숫자로만 이루어졌으면 True를 반환합니다.

isalpha() : 영문자로만 이루어졌으면 True를 반환합니다.

isnumeric() : 숫자(숫자 문자)로만 이루어졌으면 True를 반환합니다.

isdecimal() : 10진수 숫자로만 이루어졌으면 True를 반환합니다.

replace(old, new) : old 문자열을 new 문자열로 변경합니다.

**함수 정의시 매개변수의 순서 : 순서인수 > 튜플인수(\*) > 키워드인수(기본값) > 딕셔너리인수(\*\*)**



튜플 매개변수(\*매개변수) : 여러 개의 값을 하나의 튜플(tuple)로 전달받는 매개변수입니다.  /  매개변수 앞에 \*를 붙여 사용합니다.

튜플 매개변수(\*args)와 일반 매개변수를 함께 사용할 수 있습니다.  /  일반 매개변수는 앞에, 튜플 매개변수는 항상 마지막에 작성해야 합니다.



딕셔너리 매개변수(\*\*매개변수) : 여러 개의 키워드인수를 딕셔너리 형태로 받아서 가변 키워드 인수로 처리



**튜플인수 언패킹**

패킹(Packing) : 여러 개의 값을 하나의 자료형(튜플, 리스트, 딕셔너리 등)으로 묶는 것.

패킹 : 함수 정의시 매개변수 앞에 \*붙임(튜플매개변수)

언패킹(Unpacking) : 하나의 자료형(튜플, 리스트, 딕셔너리 등)에 저장된 값을 각각의 변수로 꺼내는 것.

언패킹 : 함수 호출시 매개변수 앞에 \*붙임(리스트나 튜플이 각자 요소별 매개변수로)
