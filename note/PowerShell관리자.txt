 # 1단계: 관리자 권한 PowerShell에서 실행 정책 변경 
Get-ExecutionPolicy -> RemoteSigned 인지 확인 아니면, 다음 실행
Set-ExecutionPolicy RemoteSigned

# 2단계: conda 초기화 (Anaconda Prompt에서 실행)
conda init powershell

# 3단계: PowerShell 완전히 껐다가 다시 켜기


ctrl+shift+p -> select Default profile -> command prompt 선택(ctrl+j를 하면 cmd창이 기본으로 뜨도록)

ctrl+, -> mouse 키워드 입력 -> 마우스휠을 사용할 때 ctrl키를 누르고 어쩌구저쩌구를 체크(활성화)
ctrl+, -> window.open 입력 -> window:open files in new window를 on으로 변경(두 프로젝트 이상 열기 가능)
